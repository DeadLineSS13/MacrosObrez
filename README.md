# MacrosObrez

Приложение на C# для выполнения макроса при нажатии кнопки Mouse7.

## Описание

При нажатии кнопки Mouse7 (вторая дополнительная кнопка мыши) выполняется следующая последовательность действий:

1. Сохраняется текущая позиция мыши
2. Нажимается левая кнопка мыши
3. Пауза 0.15 секунды
4. Нажимается левая кнопка мыши
5. Нажимается клавиша пробел
6. Курсор перемещается на позицию (200, 200)
7. Нажимается левая кнопка мыши
8. Курсор возвращается на сохраненную позицию

## Требования

- Windows OS
- .NET 8.0 или выше
- Мышь с кнопкой Mouse7 (боковая кнопка)

## Сборка

```bash
cd MacrosObrez
dotnet build
```

## Запуск

```bash
cd MacrosObrez
dotnet run
```

Или запустить скомпилированный exe файл:

```bash
MacrosObrez\bin\Debug\net8.0\win-x64\MacrosObrez.exe
```

## Использование

1. Запустите приложение
2. Нажмите Mouse7 для выполнения макроса
3. Нажмите Ctrl+C для выхода из приложения

## Примечание

Приложение использует низкоуровневые хуки Windows API для перехвата нажатий мыши и клавиатуры.
